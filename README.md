 # R17 Nodejs Backend Engineer Assessment - Skeleton
 This repo contains a skeleton to implement the `POST /payment-instructions`
 endpoint required by the assessment.
 ## How to run
 1. Install dependencies: `npm install`
 2. Start dev server: `npm run dev`
 3. POST to `/payment-instructions`
 Replace TODOs and implement parsing, validation and execution as specified in 
assessment.
